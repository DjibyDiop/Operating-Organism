# Documentation Translations (EN + FR + others)

This repository currently maintains primary documentation in English, with French support for core release and onboarding guidance.

## EN - Translation policy

1. English remains the canonical technical reference.
2. French must be updated for onboarding/release-critical docs.
3. Additional languages are welcome via community contributions.
4. If translations diverge, English is the source of truth.

## FR - Politique de traduction

1. L'anglais reste la reference technique principale.
2. Le francais doit etre maintenu pour les documents critiques (onboarding/release).
3. D'autres langues sont acceptees via contributions communautaires.
4. En cas de divergence, la version anglaise fait foi.

## Suggested structure

- Canonical docs:
  - `README.md`
  - `docs/PUBLIC_RELEASE_PLAYBOOK.md`
  - `docs/THIRD_PARTY_LICENSE_AUDIT.md`

- Optional language-specific mirrors:
  - `docs/i18n/fr/...`
  - `docs/i18n/es/...`
  - `docs/i18n/ar/...`

## Minimal quality bar for translated docs

1. Keep commands unchanged.
2. Keep file paths unchanged.
3. Keep warning blocks and legal statements complete.
4. Mark stale translations clearly with an update date.
