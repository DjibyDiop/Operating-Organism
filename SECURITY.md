# Security Policy

## Public Scope

`Djiby-diop/Operating-Organism` is the public-facing repository. Other related repositories and workspaces are private unless Djiby Diop explicitly publishes them.

## Reporting

Report security issues privately through the owner profile: https://github.com/Djiby-diop

Do not publish exploit details, private code, model assets, deployment material, or credentials in public issues.
